Random Fits Admin Orders Page

New files:
- admin.html
- admin.js

How to use:
1. Place a test order from checkout.html.
2. Open admin.html.
3. Enter PIN: 1234
4. Click an order to view details.
5. Print shipping labels or packing slips.

Important limitation:
This is a static website admin page.
It only sees orders saved in the same browser through localStorage.

For a real store where all customer orders appear on your admin page,
you need a backend/database later.
