Random Fits Website Google Login Setup

This update adds Google login to the WEBSITE for customers.
It is not just for admin.

New file:
- account.html
- auth.js

Updated:
- index.html
- checkout.html
- styles.css

What customers see:
- Sign in button in the website header.
- Account page.
- Google-only login button.
- No email/password login.
- After sign in, the header shows their first name.

To make Google login actually work:
1. Create a Firebase project.
2. Enable Authentication > Google sign-in.
3. Add your website domain in Firebase authorized domains.
4. Open auth.js.
5. Paste your Firebase config here:

const firebaseConfig = {
  apiKey: "PASTE_FIREBASE_API_KEY",
  authDomain: "PASTE_FIREBASE_AUTH_DOMAIN",
  projectId: "PASTE_FIREBASE_PROJECT_ID",
  appId: "PASTE_FIREBASE_APP_ID"
};

This is customer login.
Admin login is still separate in admin.js.
